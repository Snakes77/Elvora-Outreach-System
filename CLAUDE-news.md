## How to Use This Mode

This mode is for producing content anchored in live CQC data —
inspection statistics, rating trends, State of Care findings,
regulatory updates, sector workforce data.

### Workflow before drafting:
1. Fetch the relevant live URL(s) from context/cqc-intel.md
2. Extract the specific figures, findings, or regulatory changes
3. Draft content using the appropriate tone profile:
   - New regulations / enforcement → Profile 1 (Expert Authority)
   - State of Care findings → Profile 5 (Sector Insider)
   - Rating data / Outstanding tips → Profile 4 (Energising Coach)
4. Melissa's voice throughout — she is commenting as a practitioner
   who has seen these findings from the inside, not as an observer
5. Never present CQC guidance as Elvora's own advice

### What makes this content work:
- Melissa has 20+ years in the sector — she has lived these statistics
- The angle is always: "what does this mean for the registered manager
  reading this right now?"
- Specific figures beat vague claims every time
- If data is from a report older than 12 months, flag it and search for
  the latest release before using
