# Elvora Outreach Style Guide

This document defines the strict linguistic and stylistic rules for all Elvora outreach, platform content, and AI-generated copy.

## 1. UK English Standards (Strict)
All content must use **British English** spelling and grammar. 
- Use **-ise** instead of -ize (e.g., *specialise*, *organisation*, *personalisation*, *stabilise*).
- Use **-our** instead of -or (e.g., *labour*, *colour*, *honour*).
- Use **-tre** instead of -ter (e.g., *centre*).
- Use **-mme** instead of -m (e.g., *programme*).

## 2. Hyphenation Policy
ABSOLUTELY NO HYPHENS. System-wide. 
- **Incorrect**: `10-minute chat`, `well-led`, `self-audit`, `end-of-life`, `follow-up`, `3-inbox`.
- **Correct**: `10 minute chat`, `Well led`, `self audit`, `end of life`, `follow up`, `3 inbox`.
- **Note**: This applies even to compound adjectives. Clean, unhyphenated text is our signature.

## 3. Anti-Words (The GenAI Blacklist)
To ensure the outreach feels human and professional, avoid the following "AI-isms" that trigger detection and feel sterile. 

**DO NOT USE THESE WORDS:**
- **Robust**: Use "effective", "strong", "reliable".
- **Delve**: Never "delve" into data.
- **Leverage**: Use "use", "utilise", or "build upon".
- **Underscore**: Use "highlight" or "stress".
- **Embrace**: Use "adopt" or "use".
- **Foster**: Use "build", "create", or "encourage".
- **Amplify**: Use "increase" or "boost".
- **Navigate**: Use "manage" or "handle".
- **Seamless / Seamlessly**: Use "smooth" or "easy".
- **Pivotal / Vital**: Use "key", "important", or "main".
- **Innovative / Cutting-edge / State-of-the-art**: Use "modern" or "new".
- **Transformative / Revolutionise**: Too hyperbolic. Use "improve" or "change".
- **Tapestry / Nuanced**: Classic AI filler. Avoid entirely.
- **Furthermore / Moreover / Additionally**: Use "Also" or start a new sentence.
- **In summary / In conclusion**: Never signpost like a school essay.
- **A testament to...**: Avoid this flowery phrasing.
- **I hope this email finds you well**: Start with the value instead.

## 4. Tone & Persona
- **Melissa Meakin**: Warm, professional, peer-to-peer. She is a consultant, not a salesperson.
- **Direct & Honest**: Focus on the specific CQC rating (the "Quality Statement") as the anchor.
- **Value-First**: Offer the "Branded Widget" or a "10 minute chat" as a helpful resource.
