# Workflow & SOPs — Elvora Consulting

## Content Production Workflow
1. Receive brief (topic, output type, tone profile)
2. Confirm pillar alignment
3. Select tone profile
4. Draft using correct output template
5. Cross-check against anti-words.md
6. Deliver for Melissa's review
7. Approved content → add to examples/ folder

## Briefing Requirements
Before starting confirm:
- Output type (social / blog / email)
- Topic / angle
- Tone profile
- Specific stats, stories or CTAs to include
- Target audience segment
- Any timely hook

## Do Not Publish Without Melissa Confirming
- Any content referencing specific care homes by name
- Any content about ongoing regulatory or legal matters
- Any claims about specific CQC outcomes
